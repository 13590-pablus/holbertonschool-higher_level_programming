# Python - Input/Output

## Açıklama
Bu proje, Python'da dosya okuma, yazma, JSON serileştirme ve dosya yönetimi gibi temel giriş/çıkış (I/O) işlemlerini kapsar.

## Dosyalar ve Görevler
| Dosya Adı | Açıklama |
| --- | --- |
| **0-read_file.py** | UTF-8 formatındaki bir metin dosyasını okuyan ve yazdıran fonksiyon. |

## Kullanım
Fonksiyonları test etmek için şu formatta bir ana dosya kullanabilirsiniz:
```python
read_file = __import__('0-read_file').read_file
read_file("dosya_adi.txt")
```

## Gereksinimler
* Python 3.x
* Modül import etmek yasaktır (görev bazlı).
